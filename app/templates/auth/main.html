Intro
This webapp is the final project for CS50! It is a simple text and code editor.
It runs on flask and vanilla javascript with modified prism.js for code formatting.
Since it is using SQLite deployed on heroku, data is not persisted after dyno reset.
Features
Register & Login
You can register and login by filling the forms on the side panel.
Logout
If logged-in, user can logout by clicking the exit icont on top right.
Dark/light mode
You can switch between light and dark mode by clicking the moon or sun icons on the top right.
Create new post
You can create a post by clicking "Create new" on the side panel.
Delete post
Delete a post by hovering over it at the side panel and clicking the bin icon.
Editor
ENTER - hit enter for a newline field!
/command + ENTER - type in any of the following commangs at the beggining of the field and hit enter to transform the field
/p - is the default field
/h1, /h2, /h3 or (/1, /2, /3) - creates headings of respective size
/js - creates a JS formatted code field
let example = 1;
/py - creates a PY formatted code field
example = "Hello!"
/html - creates a HTML formatted code field
<h1>Hello!</h1>
/css - creates a CSS formatted code field
h1 { color: blue; }
